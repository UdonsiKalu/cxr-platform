\chapter{What is not wired (by design)}

\begin{itemize}
  \item Redis, GraphQL, Kafka, gRPC, Vault, Langfuse --- not called from Claim Studio API routes.
  \item ELK --- ships :3000 Docker logs only (not :8251 terminal dev).
  \item Optional: Compose :3000 OTel via \texttt{compose.otel-link.yaml} (separate from :8251 golden path).
\end{itemize}
